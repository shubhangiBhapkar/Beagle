# Beagle Blog Page

## 📌 Project Description
This is a simple web page about the **Beagle dog breed**.  
The page contains information about Beagles, an image section, and some trivia facts.

This project was created using **HTML and CSS** as part of an assignment.

---

## 🚀 Features
- Information section about Beagle
- Image section with a Beagle photo
- Trivia list about the Beagle breed
- Clean and simple layout

---

## 🛠️ Technologies Used
- HTML
- CSS
